# **[6주차 구현 영상]**  

# 작업명 : 인게임 구현

## 해당 요구사항 ##  

* 상하좌우 노트의 중복 생성 삭제

* 일정 비트의 맞게 노트 생성

* 일정 비트의 맞게 노트 생성


## 설명 ##

1. 노트가 플레이어 히트 박스에 동시에 닿으면 플레이어는 죽을 수 밖에 없다. 노트가 다른 방향에서 동시에 플레이어 히트박스에 닫는 상황을 제거한다. ==>20%

2. 리듬게임의 근본 살리기 ==>50%

3. 히트박스가 방향키를 이용하여 움직인다. ==> 50%

4. 플레이어가 누르고 있는 방향을 때면 게임패배된다.
